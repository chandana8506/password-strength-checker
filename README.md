# Password Strength Checker 
This is a Python-based cybersecurity mini project that checks the strength of a password.
## Features
- Checks password length
- Detects uppercase and lowercase letters
- Detects numbers
- Detects special characters
- Classifies password as Weak, Medium, or Strong

## Tools Used
- Python
- Regex
- Kali Linux / Windows

## How to Run
1. Download the file
2. Open terminal
3. Run:

python passwordchecker.py
